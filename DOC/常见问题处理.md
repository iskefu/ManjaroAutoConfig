# 常见问题处理

- 安装被锁定的问题

  ```bash
  sudo rm /var/lib/pacman/db.lck
  ```

- 无法注册数据库

  ```bash
  sudo mv /etc/lsb-release /etc/lsb-release.bak
  # 更新完成后，删除备份文件
  sudo rm -rf /etc/lsb-release.bak
  ```

- 无法提交处理（有冲突的文件）

  ```bash
  # 编辑 /etc/pacman.conf 文件，修改服务器地址
  sudo gedit /etc/pacman.conf
  ```

- 更新失败

  ```bash
  sudo pacman -Syu --ignore <package_name>
  ```
