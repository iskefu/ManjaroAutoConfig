## 用户设置
> 文件--首选项--设置
### 字体大小
```
editor.fontsize
```
### 自动保存
```
files.autoSave

onFocusChange
```
### 按下Tab键是否自动填入最佳推荐值
```
editor.tabCompletion

on
```
### 自动调整 import 语句相关顺序
```
editor.codeActionsOnSave": { 
	"source.organizeImports": true 
};
```
### 代码行号
```
editor.lineNumbers ：true;
```
